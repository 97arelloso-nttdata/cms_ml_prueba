## 0.1.0 - 2019-10-31

First release to PyPI.

### Features

- Function `cms_ml.extract_cms_features` that allows extracting aggregated features
from a folder containing raw CMS JSON files.
- Function `cms_ml.extract_cms_context` that allows extracting contextual features
from a folder containing raw CMS JSON files.
- CLI interface for easy remote execution
